### 🤝 Contributing (Contribuição)

* Este projeto foi desenvolvido como parte de um desafio de estágio. Contribuições são bem-vindas! Para contribuir:

## 1. Fork o projeto.

## 2. Crie uma feature branch (git checkout -b feature/minha-feature).

## 3. Commit suas alterações (git commit -m 'feat: Adicionada nova funcionalidade').

## 4. Push para a branch (git push origin feature/minha-feature).

## 5. Abra um Pull Request.

* Nota: Certifique-se de que seus commits seguem os padrões de mensagens de commit convencional (ex: feat:, fix:, docs:, refactor:).
